[과정]
1. 입력을 받으면서 간선을 저장하고, 진입 차수 설정
2. 진입 차수가 0인 노드를 큐에 삽입
3. 큐에서 하나씩 꺼내 방문하고, 방문 결과를  저장
4. 현재 방문하고 있는 노드와 연결된 노드의 진입 차수를 1 줄임
5. 큐가 빌 때까지 2~4까지 반복
